# Shortcut Key List
## File Operations
Ctrl+N: New File
Ctrl+O: Open File
Ctrl+S: Save File
Ctrl+Shift+S: Save As
Ctrl+W: Close Current File
Ctrl+Shift+W: Close All Opened Files
Ctrl+P: Print File
Ctrl+Shift+P: Open Command Palette

## Text Editing
Ctrl+C: Copy
Ctrl+X: Cut
Ctrl+V: Paste
Ctrl+Shift+V: Paste Without Formatting
Ctrl+Z: Undo
Ctrl+Y: Redo
Ctrl+D: Delete Current Line
Shift+Delete: Cut Selected Text
Ctrl+Delete: Delete Next Word
Ctrl+Backspace: Delete Previous Word
Ctrl+J: Join Next Line to Current Line
Ctrl+Enter: Insert New Line Below
Ctrl+Shift+Enter: Insert New Line Above
Tab: Indent Selected Text
Shift+Tab: Unindent Selected Text

## Cursor & Selection
Ctrl+A: Select All
Ctrl+Home: Jump to Document Start
Ctrl+End: Jump to Document End
Home: Jump to Line Start
End: Jump to Line End
Shift + Arrow Keys: Select Text Character by Character / Line by Line
Ctrl + Shift + Arrow Keys: Select Text Word by Word / Line by Line
Ctrl+L: Select Entire Current Line
Ctrl+Shift+L: Select All Matching Occurrences of Selected Text
Alt + Click: Multi-Cursor Edit
Ctrl+Up / Down: Scroll Page Without Moving Cursor

## Search & Replace
Ctrl+F: Find In Current File
Ctrl+H: Replace In Current File
Ctrl+Shift+F: Global Search Across All Files
Ctrl+Shift+H: Global Replace Across All Files
F3: Find Next Match
Shift+F3: Find Previous Match
Alt+R: Toggle Regular Expression Search Mode

## View & Layout
Ctrl++: Zoom In Editor
Ctrl+-: Zoom Out Editor
Ctrl+0: Reset Editor Zoom
Ctrl+\: Split Editor View
Ctrl+1 / 2 / 3: Switch Between Split Editor Panels
F11: Toggle Fullscreen Mode
Ctrl+B: Toggle Sidebar Visibility

## Comment & Format
Ctrl+/: Toggle Line Comment
Ctrl+Shift+/: Toggle Block Comment
Ctrl+Shift+I: Auto Format Document
Ctrl+[ / ]: Fold / Unfold Code Block
